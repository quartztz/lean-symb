hey! thank you for your interest in contributing to lean-symb. 

### tech
this project mainly uses [Astro](https://astro.build/) and 
[React](https://react.dev/) to build a (maximally) static website. 
some [Scala](https://www.scala-lang.org/) scripts are used to build the main
`.json` file used by the frontend.

### contributions
if you want to contribute, please fork this repository and open pull requests
against "quartztz/main". Once you've cloned, you can use the regular commands:

```
$ npm install         # install dependencies
$ npm run dev         # run locally
$ npm run build       # verify build success
$ scala process.scala # for scala
```

issues are highly appreciated before PRs, to ensure work isn't wasted and is
aligned to the goals.

please ensure the code you submit builds yada yada. use of AI is allowed, but
you should deeply understand and be able to justify any code you submit for
review. in particular, refrain from dropping a code monolith as a single PR.
the maintenance team reserves full right to ignore or request changes without reading
